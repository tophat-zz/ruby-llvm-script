Installation

This is a guide to installing ruby-llvm-script. First, before you can install ruby-llvm-script, you will need to install LLVM. You can compile it from the source or install it through Homebrew.

Installing LLVM

From Source

In order to compile LLVM, you will need to download the source from here (preferably version 3.0). Then you will need to navigate to the source in Command Prompt (Windows) or Terminal (Mac). To do this use cd:

cd path/to/llvm/source

Then configure LLVM with the following:

./configure --enable-shared --enable-jit

After that finishes, run:

make

Then finally:

make install

From Homebrew

You can also install LLVM through Homebrew:

brew install llvm --shared --jit

Installing Ruby-LLVM-Script

Then you can install ruby-llvm-script with: 

sudo gem install ruby-llvm-script

NOTE: This will install ruby-llvm as a dependency.
