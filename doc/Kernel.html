- (LLVM::Script::Library) library(name = "", opts = {}, &block)

Creates a new library or adds to it if one with the given name already exists. If this is called without a name, it returns the last created namespace in the LLVM::Script::GlobalSpace or nil if none have yet to be created. Otherwise, calls LLVM::Script::Namespace#library on the LLVM::Script::GlobalSpace.

Parameters:

  • name (Symbol, String) (defaults to: "") —

    A name for the new library.

  • block (Proc) —

    An optional block instance to build the new library with.

Returns:

  • (LLVM::Script::Library) —

    The new library.

def library(name="", opts={}, &block)
  return LLVM::Script::GlobalSpace.last if name.empty?
  return LLVM::Script::GlobalSpace.library(name, &block)
end
