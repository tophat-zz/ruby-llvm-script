- (LLVM::Script::Library) library(name = "", opts = {}, &block)

Creates a new library with the given options and an optional name. If this is called without a block and with no arguments, it returns the last created library or nil if none. If called without a block and with a name, returns the library with that name or nil if none.

Parameters:

  • name (String) (defaults to: "") —

    An optional name for the library. If you do not give one a unique id will be generated using #make_uuid.

  • opts (Hash) (defaults to: {}) —

    Options for the new library.

  • block (Proc) —

    A block with the insides of the library.

Options Hash (opts):

  • :prefix (:all, :smart, :none) — default: :smart —

    When to prefix globals with the name of the library. :smart prefixes the functions in the LLVM IR but not in ruby-llvm-script. :all does both, :none does neither.

  • :visibility (:public, :private) — default: :public —

    The default visibility of globals in the library.

Returns:

  • (LLVM::Script::Library) —

    The new library.

See Also:

  • LLVM::Script::Library#initialize

def library(name="", opts={}, &block)
  unless block_given?
    return LLVM::Script::Library.lookup(name)
  else
    return LLVM::Script::Library.new(name, opts, &block)
  end
end

- (LLVM::Script::Program) program(name = "", opts = {}, &block)

Creates a new program with the given options and an optional name. If this is called without a block and with no arguments, it returns the last created program or nil if none. If called without a block and with a name, returns the program with that name or nil if none.

Parameters:

  • name (String) (defaults to: "") —

    An optional name for the program. If you do not give one a unique id will be generated using #make_uuid.

  • opts (Hash) (defaults to: {}) —

    Options for the new program, same as those for a library (except :prefix is ALWAYS :none).

  • block (Proc) —

    A block with the insides of the program.

Returns:

  • (LLVM::Script::Program) —

    The new program.

See Also:

  • LLVM::Script::Program#initialize
