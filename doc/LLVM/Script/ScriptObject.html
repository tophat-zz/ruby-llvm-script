Overview

This class does nothing on non-Ruby 1.9.x platforms.

On Ruby 1.9.x, however, it inherits BasicObject. This allows it to ignore default Kernel methods unless the method is missing. This allows you not to have to self.methodname syntax when a Library or Program function name conflicts with a Kernel function. An example of this is in the hello world program.
