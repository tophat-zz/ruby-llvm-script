Overview

This class is Ruby version dependent.

Ruby 1.8.x

It just provides a method to make a Uuid (Universally unique identifier).

Ruby 1.9.x

It inherits BasicObject, allowing it to ignore default Kernel methods unless the method is missing. This allows you not to have to self.methodname syntax when a Library or Program function name conflicts with a Kernel function. An example of this is in the hello world program in the README.
